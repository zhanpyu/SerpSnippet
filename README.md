# SerpSnippet

Preview and validate your page metadata before it goes live — instantly check how your title tags and meta descriptions will render in Google search results.

## What it does

SerpSnippet gives you a real-time SERP preview as you type, with live character count validation against SEO best practices. Useful for content teams and SEO audits to catch truncation and length issues before publishing.

## Features

- Real-time Google SERP preview (desktop & mobile)
- Title tag validation — highlights too short (<30 chars) or too long (>60 chars)
- Meta description validation — highlights too short (<70 chars) or too long (>155 chars)
- Character counters with color-coded feedback
- Clean, minimal UI — no install, runs in the browser

## Tech Stack

- React + TypeScript
- Tailwind CSS
- Vite

## Getting Started

```bash
cd SerpSnippet/project
npm install
npm run dev
```
