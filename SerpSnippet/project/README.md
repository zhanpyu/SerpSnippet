# Aperçu de Résultat Google (Google SERP Preview)

Cette application permet de prévisualiser l'apparence d'une page web dans les résultats de recherche Google.

## Installation

```bash
npm install
```

## Démarrage

```bash
npm run dev
```

L'application sera accessible à l'adresse indiquée dans le terminal.

## Fonctionnalités

- Prévisualisation en temps réel du résultat de recherche
- Validation de la longueur du titre (50-60 caractères)
- Validation de la méta description (150-160 caractères)
- Interface entièrement en français
- Conseils SEO intégrés
